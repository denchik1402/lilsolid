### PWA (Progressive Web App) — что это?

**PWA** — это сайт, который можно установить как приложение на телефон или компьютер (иконка на рабочем столе, запуск в отдельном окне).

**Что нужно:**
- **manifest.json** — название, иконки, цвета, режим отображения (standalone)
- **Service Worker** — скрипт, который кэширует страницы для офлайн-доступа

**Зачем:** пользователь может «Установить приложение» в браузере, открывать магазин как приложение, частично работать без интернета (кэш). Удобно для мобильных.

---

### Тесты (pytest) — для чего?

**Тесты** проверяют, что код работает как задумано, особенно после изменений.

**Зачем:**
- При рефакторинге — убедиться, что ничего не сломалось
- При добавлении фич — проверить API, маршруты, логику
- Автоматическая проверка вместо ручного прогона по сайту

**Пример:** тест «GET /api/search-suggestions?q=IQOS возвращает список» — если кто-то сломает поиск, тест упадёт.

## 2. SEO: что сделать для высоких позиций в поиске

### 2.1 Уже реализовано ✓

- robots.txt с Sitemap
- sitemap.xml (главная, каталог, категории, товары, статические страницы)
- Schema.org: WebSite, Organization, LocalBusiness, Product, BreadcrumbList, FAQPage, ItemList
- Open Graph (og:title, og:description, og:image, og:url)
- meta description, meta keywords
- canonical URL
- Хлебные крошки (BreadcrumbList)

### 2.2 Рекомендации по SEO (приоритетные)

#### A. Технические

| Действие | Описание |
|----------|----------|
| **ЧПУ для товаров** | Сейчас `/product/123`. Лучше `/product/iqos-iluma-i-one-midnight-black` или `/catalog/terea-sticks/iqos-iluma-i-one`. Поисковики лучше индексируют URL с ключевыми словами. |
| **lastmod в sitemap** | Добавить `<lastmod>` для товаров и категорий — поисковики чаще переиндексируют изменённые страницы. |
| **Абсолютные URL в canonical** | Убедиться, что canonical всегда с https и доменом (уже в основном есть). |
| **hreflang** | Если планируется англоязычная версия — добавить hreflang для мультиязычности. |

#### B. Контент

| Действие | Описание |
|----------|----------|
| **Уникальные meta для категорий** | Сейчас категории наследуют base. Добавить block meta_description в catalog.html с текстом вида: «Купить [название категории] в LIL STORE. Оригинальная продукция, доставка 1–2 дня.» |
| **Alt у всех изображений** | Проверить, что у картинок товаров есть осмысленный alt (название товара, ключевые слова). |
| **Заголовки H1–H6** | Одна H1 на страницу, логичная иерархия. На товаре — H1 с названием. |
| **Внутренняя перелинковка** | Ссылки «Похожие товары», «В этой категории», «Хиты» — уже есть. Добавить блок «Популярные запросы» на главной. |

#### C. Скорость и Core Web Vitals

| Действие | Описание |
|----------|----------|
| **Оптимизация изображений** | WebP, сжатие, правильные размеры (srcset для респонсива). |
| **Отложенная загрузка** | `loading="lazy"` уже есть. Добавить для изображений ниже первого экрана. |
| **Критический CSS** | Инлайнить минимальный CSS для above-the-fold. |
| **Preload шрифтов** | Если используются кастомные шрифты. |

#### D. Расширенная разметка

| Действие | Описание |
|----------|----------|
| **Product: несколько изображений** | В Schema.org Product добавить массив `image` со всеми фото товара. |
| **Product: oldPrice** | Если есть скидка — добавить `priceSpecification` с oldPrice для rich snippets. |
| **Review в отдельной разметке** | При наличии отзывов — разметка Review для звёзд в выдаче. |
| **VideoObject** | Если появятся видео товаров — разметка для видео в поиске. |

#### E. Внешние инструменты

| Действие | Описание |
|----------|----------|
| **Google Search Console** | Подключить сайт, отправить sitemap, отслеживать индексацию и ошибки. |
| **Яндекс.Вебмастер** | Аналогично для Яндекса. |
| **Яндекс.Метрика / Google Analytics** | Аналитика, цели, поведение пользователей. |
| **Проверка в PageSpeed Insights** | Регулярно проверять LCP, FID, CLS и устранять проблемы. |

### 2.3 Быстрые победы (можно сделать сразу)

1. **lastmod в sitemap** — дата изменения товара/категории.
2. **Уникальные meta для категорий** — описание и ключевые слова.
3. **Product Schema: oldPrice** — для товаров со скидкой.
4. **Product Schema: image[]** — все изображения товара.
5. **Preconnect к CDN** — ускорение загрузки Bootstrap/Font Awesome.
