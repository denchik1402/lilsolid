# План SEO-улучшений

Оставшиеся пункты.

**Пояснения:**
- **Поле alt в админке** — сейчас у картинок товаров alt = название товара. Поле в админке позволит задать свой текст (например, «IQOS Iluma Prime синий — фото»).
- **WebP** — формат изображений, даёт файлы на 25–35% меньше при том же качестве.
- **srcset** — браузер выбирает подходящий размер картинки (на телефоне — меньше, на десктопе — больше).
- **Nginx** — настройки веб-сервера: сжатие, кэш, HTTP/2.

---

## Технические

- [ ] **Поиск** — рассмотреть `noindex` при пустом запросе (опционально)
- [ ] **Product Schema** — при наличии: gtin, mpn (опционально)
- [ ] **og:image** — проверить размер (рекомендуется 1200×630)
- [x] **Поле alt в админке** — отдельное поле для alt-текста изображений товаров (сейчас используется название товара)

---

## Производительность

- [ ] **Preload** — критичных ресурсов (если нужно)
- [ ] **WebP** — формат изображений с меньшим размером при том же качестве
- [ ] **srcset** — несколько размеров картинок для разных экранов (мобильный получает меньший файл)
- [ ] **Lazy loading** — проверка (уже есть на карточках)
- [ ] **Сжатие** — оптимизация при загрузке в админке

## Сервер (Nginx)

- [ ] **Brotli/Gzip** — сжатие ответов сервера
- [ ] **Cache-Control** — кэширование статики (картинки, CSS, JS)
- [ ] **HTTP/2** — ускорение загрузки
