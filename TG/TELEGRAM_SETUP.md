# Настройка Telegram-бота магазина

## Роли пользователей

| Роль | Видит меню | Возможности |
|------|------------|-------------|
| **Пользователь** | User | Каталог, корзина, оформление заказа |
| **Курьер** | User + Courier | Заказы с пагинацией, маршрут в Яндекс.Картах, отметка выполненным |
| **Boss** | User + Courier + Boss | Управление сайтом, статистика, прибыль, модерация отзывов |
| **Admin** | Всё | Назначение курьеров и боссов |

**Админ по умолчанию:** пользователь с ником `@denchik1402` при первом `/start` получает роль admin.

## Шаг 1: Создание бота

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Введите имя (например: `LIL STORE`)
4. Введите username (например: `iqos_store_bot`)
5. Скопируйте **токен**

## Шаг 2: Настройка config.py

В **корне проекта** (my_shop) создайте или отредактируйте `config.py`:

```python
TELEGRAM_BOT_TOKEN = 'ваш_токен'
TELEGRAM_CHAT_ID = ''  # не нужен: добавьте бота в группу и напишите /start
TELEGRAM_DEFAULT_ADMIN = 'denchik1402'
ILUMA_XLSX_PATH = r'c:\путь\к\iluma.xlsx'  # для загрузки себестоимости (один раз)
SITE_URL = 'http://ваш-сайт.ru'
```

## Шаг 3: Запуск

**Из корня проекта (my_shop):**

```bash
python TG/telegram_bot.py
```

Или на Windows: `TG\run_bot.bat`

Бот должен работать постоянно (на сервере — через systemd/supervisor).

## Раздельный деплой (бот на одном устройстве, сайт на другом)

Скопируйте папку `TG` и следующие файлы из корня проекта:
- `app.py`, `models.py`, `extensions.py`, `config.py`
- `shop.db` (или настройте удалённую БД)
- папку `static/images/products` (для фото товаров в боте)

На устройстве с ботом запускайте из папки, где лежит `app.py` и `TG`:
```bash
python TG/telegram_bot.py
```

Убедитесь, что `config.py` указывает на тот же `shop.db` или на общую БД.

## Функции бота

### Пользователь (все)
- 🛒 Каталог — просмотр товаров, добавление в корзину
- 🛍 Корзина — изменение количества, оформление заказа
- 📦 Оформление заказа — ввод имени, телефона, адреса

### Курьер
- 🚚 Заказы — пагинация по 5, отметка «Выполнен»
- 🗺 Маршрут — построение маршрута в Яндекс.Картах по адресам заказов

### Boss
- 🛠 Управление сайтом — ссылка на админку, редактирование товаров
- 📦 Заказы — просмотр с пагинацией
- 📊 Статистика — заказы, выручка, отзывы
- 💰 Прибыль — расчёт по iluma.xlsx (себестоимость)
- 💬 Модерация отзывов — одобрение/отклонение

### Admin
- ⚙️ Назначение ролей — курьер, boss, admin для пользователей

## Модерация отзывов

Все отзывы проходят модерацию. Boss и Admin могут одобрять/отклонять в меню Boss → Отзывы на модерации.

## Уведомления

Уведомления о заказах и отзывах отправляются в чат, где вы написали боту:
1. Напишите боту в личку `/start` — уведомления пойдут вам в личку
2. Или добавьте бота в группу и напишите `/start` — уведомления пойдут в группу
3. Или Boss/Admin: `/set_notify` в нужном чате — явно задать чат для уведомлений
4. TELEGRAM_CHAT_ID в config не нужен
