# Изображения

## Баннеры и блоки (управление в админке)

Баннеры карусели и блоки «Премиальные устройства» загружаются через админку: **Админ → Баннеры**.

**Рекомендуемые размеры:**
- **Карусель главной**: 800×450 px или 1200×600 px (ширина × высота)
- **Блок 1** (левый большой): 600×400 px
- **Блоки 2–3** (правые): 400×200 px

Файлы сохраняются в `static/images/banners/`.

## Изображения товаров

## Структура папок

```
static/images/products/
├── Devices/
│   ├── Iqos Iluma i Standart Seletti/
│   │   └── iqos_iluma_seletti.webp    ← любой файл .jpg, .png, .webp
│   └── IQOS Iluma i Standart Vivid Terracotta/
│       └── iqos_iluma_i_vivid_terracotta.webp
└── Sticks/
    └── ...
```

## Важно

1. **Имя папки** должно **точно** совпадать с названием товара в каталоге (включая регистр и пробелы).
2. **Форматы**: `.jpg`, `.jpeg`, `.png`, `.webp`
3. **Путь**: `static/images/products/Devices/Имя товара/файл.jpg`

## Если изображения не отображаются

Запустите скрипт синхронизации:

```bash
python fix_product_images.py
```

Скрипт найдёт первый подходящий файл в папке товара и обновится запись в базе данных.

## Проверка

- Папка `Iqos Iluma i Standart Seletti` (обратите внимание: **Iqos** с маленькой q)
- Папка `IQOS Iluma i Standart Vivid Terracotta` (обратите внимание: **IQOS** и **Standart** с одной t)
